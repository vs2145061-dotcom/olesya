# Визуал и темы оформления (день / ночь)

Версия визуала: v2.1. Изменения затрагивают только `02-приложение/index.html`
(вся вёрстка/стили/логика — в одном файле, по архитектуре проекта). Логика, тесты
и бэкенд не менялись — **все 149 автотестов проходят**.

## Что сделано

На основе присланных референсов (насыщенный фиолетово-сиреневый UX/UI + чистые
светлые мессенджер-экраны) собраны **две полноценные темы** с автопереключением.

### 🌙 Тёмная тема (ночь) — по умолчанию
- Глубокий фиолетово-чёрный фон с неоновым свечением (как в референсах).
- Сообщения «от меня» — яркий сиренево-фиолетовый градиент; «входящие» — тёмно-фиолетовые карточки.
- Экраны звонка и блокировки — эффектный тёмно-фиолетовый «погружной» фон.

### ☀️ Светлая тема (день)
- Чистый лавандово-белый фон, белые поверхности, мягкие тени.
- Тот же сине-фиолетовый бренд-акцент; входящие пузыри — белые с тонкой обводкой/тенью.
- Повышенная контрастность текста и акцентов для дневного света.

### 🔄 Автопереключение и управление
- **Авто** — приложение следует системной теме устройства (`prefers-color-scheme`)
  и переключается между днём/ночью на лету (в т.ч. по системному расписанию).
- **Ручной выбор** в «Настройки → Оформление → Тема оформления»: Авто / Светлая / Тёмная.
- **Быстрый тумблер** (☀️/🌙) в шапке списка чатов — мгновенно переключает тему.
- Выбор сохраняется (`localStorage: om_theme`).
- Тема применяется **до отрисовки** (нет вспышки чужой темы при запуске).
- Цвет системной строки (`meta theme-color`) обновляется под активную тему.

## Технически
- Все цвета — на CSS-переменных: `:root` / `:root[data-theme="dark"]` (ночь) и
  `:root[data-theme="light"]` (день). Раньше зашитые тёмные цвета вынесены в переменные
  (`--input-focus`, `--immersive`, фоновое свечение и т.д.).
- Контроллер тем: `applyTheme` / `resolveTheme` / `setThemePref` + слушатель `matchMedia`
  для режима «Авто».
