## 1. Что это

**Olesya Messenger** — кроссплатформенный мессенджер по принципу «один HTML-файл = всё приложение».
Один файл `index.html` (~166 КБ) работает одновременно как:

- **Веб-приложение (PWA)** — браузер, устанавливается как иконка
- **Android-приложение** — тот же HTML в обёртке Expo WebView (APK собран, работает)
- **iOS-приложение** — тот же код (готов к сборке, см. iOS-ИНСТРУКЦИЯ.md)
- **Десктоп** — любой браузер

Обновление одного файла на хостинге обновляет все платформы сразу.

## 5. Функциональность (всё работает, 149 тестов)

**Вход:** SMS через SMSC (РФ) и Firebase (мир), сессии 30 дней, 2FA, защита от перебора.

**Синхронизация:** переписка между устройствами, поиск контакта по номеру, история чата.

## 8. Как продолжить

1. **Обновить интерфейс:** редактировать `index.html` → залить на GitHub → приложения подхватят сами
2. **Изменить сервер:** редактировать `worker-paste.js` → вставить в Cloudflare → Deploy
3. **Запустить тесты:** `node test_otp2.js` (нужен `npm install jsdom`) → 149/149
4. **Собрать приложение:** `eas build -p android --profile preview` (Android) или см. iOS-ИНСТРУКЦИЯ.md

⚠️ `index.html` содержит реальные ключи Firebase и Worker URL. Перед публичной выкладкой архива — вычистить.
