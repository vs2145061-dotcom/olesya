# Премиальный визуал — глянцевые 3D-иконки

Версия: v2.7. Изменения только в `02-приложение/index.html` (CSS + разметка иконок).
**Все 149 автотестов проходят.** Логика, бэкенд и тесты не затрагивались.

## Анализ референсов

Иконки-референсы (соцсети) объединяет один узнаваемый «дорогой» стиль:

1. **Скруглённый «пузырь»** — идеальный круг (или squircle для плиток), без острых углов.
2. **Диагональный градиент** — светлее в левом верхнем углу (≈32% / 24%), темнее к правому
   нижнему; источник света всегда сверху-слева.
3. **Стеклянный блик** — мягкий светлый овал в верхней части, как отражение на капле/стекле.
4. **Внутренний объём** — тёмная тень снизу-внутри и светлая «кромка» сверху-внутри
   (`inset`-тени) создают ощущение выпуклости.
5. **Мягкая отбрасываемая тень** — размытая тень под объектом «приподнимает» его над фоном.
6. **Белый глиф по центру** — простой, толстый, с большим визуальным запасом; всегда поверх
   градиента и блика.

## Как это реализовано (CSS-«рецепт» объёма)

Базовый класс `.orb` собирает все шесть признаков в чистом CSS — без растровых картинок,
поэтому масштабируется без потери чёткости и не утяжеляет файл:

```css
.orb{
  border-radius:50%;
  box-shadow:
    inset 0 -4px 7px rgba(0,0,0,.30),     /* нижняя внутренняя тень — объём */
    inset 0 3px 5px rgba(255,255,255,.5), /* верхняя внутренняя кромка — блик */
    0 9px 18px -6px rgba(0,0,0,.5);       /* мягкая тень под иконкой */
}
.orb::after{ /* стеклянный блик-отражение сверху */
  background:linear-gradient(rgba(255,255,255,.85),rgba(255,255,255,0));
}
.orb>svg{ z-index:1 } /* белый глиф поверх градиента и блика */
```

Цвет каждой иконки задаёт диагональный `radial-gradient(120% 120% at 32% 24%, …)`:

| Класс       | Назначение         | Градиент              |
|-------------|--------------------|-----------------------|
| `.o-chats`  | Чаты               | фиолетовый → индиго    |
| `.o-calls`  | Звонки             | мятный → зелёный       |
| `.o-book`   | Телефонная книга   | сиреневый → синий      |
| `.o-fav`    | Избранное          | золотой → оранжевый    |
| `.o-brand`  | Логотип «O»        | лавандовый → индиго    |

## Где применён стиль

- **Лицевая страница (хаб)** — карточки Чаты / Звонки / Телефонная книга / Избранное:
  глянцевые «пузыри» `.orb` с белыми SVG-глифами вместо emoji; сами карточки получили
  верхний световой блик (`.home-card::before`) и многослойные тени — выглядят как стекло.
- **Логотип «O»** — отрисован тем же глянцевым орбом (`.orb.brand.o-brand`).
- **Аватары** (списки чатов, шапка, профиль) — добавлен стеклянный блик `.avatar::after`
  и мягкая тень: лица/emoji смотрятся объёмно, как 3D-стикеры.
- **Кнопки действия** — FAB «＋», кнопка отправки, кнопки звонка (принять/завершить) и
  основные `.btn` получили градиент, верхний блик и тень — единый дорогой язык интерфейса.

## Принципы, которые сохранили продукт «дорогим»

- Один источник света на весь интерфейс (сверху-слева) — ничего не «спорит».
- Глянец сделан полупрозрачными белыми бликами, поэтому **одинаково работает и в ночной,
  и в дневной теме** — правила не привязаны к теме.
- Никаких картинок-иконок: всё на CSS + SVG → чёткость на любом DPI, вес файла не вырос.
