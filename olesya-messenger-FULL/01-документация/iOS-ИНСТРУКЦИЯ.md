# iOS без Mac — тест на своём iPhone

Цель: установить Olesya Messenger на ваш iPhone, не имея Mac.
Путь: EAS собирает .ipa в облаке → TestFlight устанавливает на телефон.

---

## Что понадобится

- Apple Developer Program — **$99/год** (карта у вас есть)
- Аккаунт Expo (уже есть — вы делали Android)
- Node.js + EAS CLI на компьютере (уже установлены)
- iPhone с установленным приложением **TestFlight** (бесплатно в App Store)
- **Mac НЕ нужен**

---

## ЭТАП 1 — Регистрация Apple Developer (один раз)

> ⚠️ Это занимает время: Apple проверяет личность от нескольких часов до 2 суток,
> иногда просит фото документа. Начните заранее.

1. Откройте **https://developer.apple.com/programs/enroll/**
2. Войдите своим Apple ID (или создайте новый)
3. Выберите тип аккаунта: **Individual** (Физическое лицо)
   - Для Individual НЕ нужен D-U-N-S номер (он только для компаний)
4. Заполните данные, оплатите $99 картой
5. Дождитесь письма от Apple о подтверждении (Welcome to the Apple Developer Program)

Пока ждёте подтверждения — переходите к Этапу 2, код можно собрать заранее.

---

## ЭТАП 2 — Подготовить проект

1. Замените `eas.json` в папке проекта на новый (из этой выдачи)
2. Откройте терминал в папке `olesya-native`
3. Убедитесь, что вошли в Expo:
   ```bash
   eas whoami
   ```
   Если «Not logged in» — задайте токен (как для Android):
   ```powershell
   $env:EXPO_TOKEN="ваш_токен_expo"
   ```

---

## ЭТАП 3 — Собрать iOS-приложение в облаке

```bash
eas build -p ios --profile production
```

Что произойдёт:
1. EAS спросит: **«Do you want to log in to your Apple account?»** → **Y**
2. Введите Apple ID и пароль (тот, на который оформляли Developer Program)
3. Если включена двухфакторная аутентификация — введите код с другого устройства Apple
4. EAS **автоматически создаст** сертификат распространения и provisioning profile — ничего вручную делать не нужно
5. Сборка уйдёт в облако на **10–20 минут** (компьютер можно закрыть)
6. По завершении появится ссылка на `.ipa` и сборка будет на **expo.dev → Builds**

> Если Apple Developer ещё не подтverждён — этот шаг выдаст ошибку про membership.
> Дождитесь письма от Apple и повторите.

---

## ЭТАП 4 — Загрузить в TestFlight

```bash
eas submit -p ios --latest
```

Что произойдёт:
1. EAS спросит Apple ID — введите тот же
2. EAS **сам создаст запись приложения** в App Store Connect (если её нет)
3. Загрузит `.ipa` в App Store Connect
4. Через 5–15 минут сборка появится в TestFlight (Apple её обрабатывает)

> Первая обработка билда в App Store Connect может занять до получаса — это нормально.

---

## ЭТАП 5 — Добавить себя в тестировщики

1. Откройте **https://appstoreconnect.apple.com**
2. **Мои приложения** → **Olesya Messenger** → вкладка **TestFlight**
3. Раздел **Внутреннее тестирование** (Internal Testing) → нажмите **+** у группы
4. Добавьте свой Apple ID как тестировщика (вы уже Account Holder, так что доступны сразу)
5. Выберите загруженный билд

> Внутренним тестировщикам **не нужен** Beta App Review — можно ставить сразу.
> (Review требуется только для внешних тестировщиков.)

---

## ЭТАП 6 — Установить на iPhone

1. На iPhone установите приложение **TestFlight** из App Store (если ещё нет)
2. Войдите в TestFlight тем же Apple ID
3. Откройте TestFlight → увидите **Olesya Messenger** → нажмите **Установить**
4. Иконка появится на экране — запускайте, тестируйте

Готово — приложение на вашем iPhone.

---

## Если что-то пойдёт не так

**«You have no team associated with your Apple account»**
→ Apple Developer Program ещё не активирован. Дождитесь письма от Apple.

**Сборка падает на этапе Fastlane / signing**
→ Откройте expo.dev → Builds → нажмите на сборку → читайте логи. Пришлите мне ошибку.

**Двухфакторная аутентификация мешает в терминале**
→ Заведите App-Specific Password: appleid.apple.com → вход → Безопасность →
   Пароли приложений → создать → используйте его вместо обычного пароля.

**В TestFlight приложение не появилось**
→ Подождите до 30 минут (Apple обрабатывает билд). Проверьте App Store Connect →
   TestFlight → статус билда должен стать «Готов к тестированию».

## Альтернатива — без TestFlight (быстрее, только ваш телефон)

Если не хотите возиться с App Store Connect, можно установить напрямую,
зарегистрировав конкретный iPhone:

1. Зарегистрировать устройство (откроется ссылка, перейдите по ней с iPhone):
   ```bash
   eas device:create
   ```
2. Собрать сборку для зарегистрированных устройств:
   ```bash
   eas build -p ios --profile preview
   ```
3. По завершении откройте ссылку/QR на iPhone → установите профиль → установится приложение

Минус этого пути: каждый новый iPhone нужно регистрировать отдельно (UDID) и пересобирать.
Плюс: не нужен App Store Connect. Для одного-двух своих устройств — удобно.

> Apple Developer Program ($99/год) нужен и для этого пути тоже.

## После успешного теста — публикация в App Store

Когда протестируете и захотите выложить для всех:
1. App Store Connect → ваше приложение → заполнить карточку
   (описания готовы в `stores/ios/` из архива)
2. Скриншоты для iPhone 6.9" и 6.5"
3. Отправить на ревью → ожидание 1–3 дня

Подробности — в `BUILD-APPS.md` из архива проекта.
