# Фоны чатов и формы видео-кружков (по настроению)

Версия визуала: v2.2. Изменения только в `02-приложение/index.html`.
Логика и бэкенд не затронуты — **все 149 автотестов проходят**.

## 🖼️ Фон чатов

В «Настройки → Оформление → Фон чатов» можно выбрать оформление области сообщений
для всех чатов. Доступно **7 готовых стилей**, «Стандартный» (без фона) и загрузка
своего фото:

| Стиль | Описание |
|---|---|
| ⭐ Звёздный | Тёмное ночное небо со звёздами |
| ☀️ Солнечный | Тёплый персиково-жёлтый градиент с солнцем |
| ◼️ Мозаика | Геометричная фиолетовая шахматка |
| 🌌 Аврора | Мягкое северное сияние (зелёно-фиолетовое) |
| 🌊 Океан | Синий градиент с лёгкими «волнами» |
| 🌇 Закат | Розово-оранжево-фиолетовый закат |
| 🟪 Неон-сетка | Кибер-сетка на тёмном (в стиле референсов) |

**Своё фото** — загрузка изображения с устройства: фото автоматически уменьшается
(до 1280 px, JPEG) и сохраняется локально (`om_chatbg_photo_<id>`), чтобы не
переполнять хранилище. Применяется как `cover` по центру.

- Выбор фона — глобальный (для всех чатов), сохраняется (`om_chatbg_<id>`).
- В окне выбора у каждого варианта живое превью стиля.
- Фоны корректно работают и в светлой, и в тёмной теме; пузыри сообщений остаются
  читаемыми над любым фоном.

## 🎭 Форма видео-кружка по настроению

Перед записью видео-кружка можно выбрать его **форму**, которая передаёт настроение:

| Форма | Настроение |
|---|---|
| ⚪ Кружок | нейтрально (по умолчанию) |
| ❤️ Сердечко | тепло |
| ☀️ Солнышко | радость |

**Как пользоваться:**
- **Одиночный тап** по кнопке кружка — обычная запись (форма «кружок»).
- **Двойной тап** по кнопке кружка — всплывает выбор формы. После выбора формы
  сразу начинается запись в выбранной форме.

Выбранная форма применяется и к кольцу записи (предпросмотр камеры обрезается по
форме), и к отправленному сообщению; у сообщений с формой «сердечко»/«солнышко»
показывается значок настроения (❤️/☀️). Форма действует на одну запись — следующий
одиночный тап снова даёт обычный кружок.

*Технически:* формы заданы через SVG `clipPath` (objectBoundingBox), пузырь
видео-кружка сделан полностью прозрачным, чтобы фигура «парила» на фоне чата.
Форма передаётся получателю в реальном времени через шину событий.

*Надёжность:* добавлена защита от утечки камеры — если запись отменена, пока
запрашивался доступ к камере, медиапоток корректно освобождается.
