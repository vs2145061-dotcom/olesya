## Что осталось

### 1. iOS — запись в Apple Developer (приоритет, если нужен iOS)
**Блокер:** ошибка «An unknown error occurred» на странице Enroll Apple Developer.
Это на стороне Apple, не приложения. Решения — в ИСТОРИЯ-ОШИБОК-ПОЛНАЯ.md (E7) и iOS-ИНСТРУКЦИЯ.md.
Кратко: проверить двухфакторную аутентификацию Apple ID (причина №1), добавить способ оплаты,
пробовать приложение Apple Developer на iPhone, при повторе — Apple Developer Support.
Код iOS полностью готов (App.js, app.json, eas.json исправлен под реальное устройство).

### 2. Медиа в R2 (опционально)
Сейчас фото/голос передаются как base64 — работает, но тяжело.
В воркере готов `/media` для R2-хранилища. Включается: `wrangler r2 bucket create olesya-messenger-media`
(нужен Node.js + wrangler локально). Не блокирует работу.

### 3. Перевод через Claude (опционально)
В воркере готов `/translate` (прокси Anthropic API). Без секрета `ANTHROPIC_API_KEY`
перевод идёт через резерв (Google/словарь). Включается: `wrangler secret put ANTHROPIC_API_KEY`.

### 4. Публикация
- Google Play: $25 разово, `eas build -p android --profile production` → .aab
- App Store: $99/год, после enrollment, см. iOS-ИНСТРУКЦИЯ.md
- Описания и материалы — в инструкциях

### 5. Регистрация имени отправителя SMSC (опционально)
Сейчас SMS с общего номера SMSC (работает сразу). Для имени «Olesya» вместо номера —
отдельная регистрация у SMSC (несколько дней, документы). Для запуска не нужно.
