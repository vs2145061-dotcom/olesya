## 📰 Telegram в «Социальных сетях» теперь — новостной канал

Раньше Telegram открывал вашу страницу-мессенджер. Теперь это **новостной канал**, а не
«мессенджер внутри мессенджера»:
- В папке «Социальные сети» Telegram отмечен значком **НОВОСТИ** и кнопкой **«Добавить
  канал»** (остальные соцсети — как раньше, «Подключить» свою страницу).
- Вы указываете публичный канал (например `@meduzalive` или `t.me/rian_ru`).
- Открывается **лента постов канала в стиле Olesya** (карточки с текстом, картинкой и
  временем), тап по посту — читалка с возможностью «✨ Пересказать ИИ» и перейти к посту.
- Технически: сервер парсит публичную страницу `https://t.me/s/<канал>` — новый эндпоинт
  **`GET /tg?channel=`** (в `worker.js` и `worker-paste.js`); посты приходят как новости.
  Без подключённого сервера показывается понятная подсказка.

То есть Telegram используется именно как источник новостей, а не как чат.

  сгенерированы из фирменной иконки;
- `assets/www/index.html` — актуальная сборка мессенджера (локальный резерв).

### Команды пересборки (запускаются у вас — сборка идёт в облаке Expo)

```bash
cd 05-нативное-приложение
npm install                       # поставит зависимости (incl. expo-contacts, webview)
npx expo install --fix            # выровнять версии под вашу версию Expo SDK (рекомендуется)
npm install -g eas-cli            # если eas ещё не установлен
eas login                         # вход в ваш аккаунт expo.dev
eas build -p android --profile preview      # APK для теста на телефон
# релизные сборки:
eas build -p android --profile production    # AAB для Google Play
eas build -p ios --profile production        # IPA для App Store
```

> **Почему сборку запускаете вы:** `eas build` выполняется в облаке Expo и требует входа в
> ваш аккаунт (`eas login` / `EXPO_TOKEN`). Из этой среды я не могу авторизоваться в вашем
> аккаунте Expo — поэтому подготовил проект «под ключ», а команду сборки нужно выполнить вам
> (5–15 минут, ссылка на APK/AAB/IPA придёт на email и появится на expo.dev → Builds).
