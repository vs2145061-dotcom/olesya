# Лицевая страница, телефонная книга и звонки

Версия: v2.3. Изменения только в `02-приложение/index.html` (плюс 4 ассерта в тестах
под новый экран входа). **Все 149 автотестов проходят.**

## 🏠 Лицевая страница (хаб)

После входа открывается презентабельная главная страница, с которой пользователь
выбирает раздел:

- **💬 Чаты** — со счётчиком непрочитанных сообщений;
- **📞 Звонки** — история вызовов;
- **📇 Телефонная книга** — контакты;
- **🔖 Избранное** — сохранённые сообщения.

Сверху — приветствие по имени, аватар (открывает настройки), переключатель темы и
шестерёнка настроек. В каждый раздел можно зайти с хаба, а вернуться — кнопкой
«домой» в шапке раздела. Карточки — крупные градиентные плитки в фирменном стиле,
красиво смотрятся в обеих темах.

## 📇 Телефонная книга

Экран «Телефонная книга» (карточка на хабе или иконка телефона):

- **Поиск по имени и номеру** — фильтрует ваши контакты.
- **Добавление по номеру** — введите номер (форматы `+7…`, `8…`, `7…` распознаются);
  если такого контакта ещё нет, появляется кнопка **«Добавить …»**, которая ищет
  пользователя в мессенджере (по номеру регистрации) и добавляет его.
- **Импорт из телефона** — кнопка «Импорт из телефона» открывает системный выбор
  контактов устройства (Contact Picker API). Для каждого выбранного контакта берётся
  его **имя и номер**, номер проверяется в мессенджере, и зарегистрированные
  контакты добавляются с именем из телефонной книги. Если браузер не поддерживает
  доступ к контактам — подсказка добавить вручную по номеру.
- **Контакты** — список с аватаром, именем, номером, индикатором онлайн. Нажатие —
  открывает чат. Справа — кнопки **аудиозвонок 📞**, **видеозвонок 🎥** и
  **переименование ✎** (своё имя контакта хранится локально, как в обычной книге).

> Поиск нового человека идёт по **номеру регистрации** (серверный поиск). Поиск по
> **имени** работает по вашим уже известным/импортированным контактам — это
> ограничение приватности децентрализованной модели (сервер не отдаёт людей по имени).

## 📞 Звонки

Экран «Звонки» — история вызовов: для каждого показаны имя, направление
(↗ исходящий / ↙ входящий / ⤬ пропущенный), тип (📞 аудио / 🎥 видео), время и
длительность. Нажатие на запись или кнопки справа — **обратный звонок**. Кнопка `+`
сверху — выбрать контакт из книги. История ведётся автоматически и хранится локально
(`om_calls_<id>`, последние 100).

## Технические детали

- Новые экраны: `#scrHome`, `#scrContacts`, `#scrCalls`. Вход (`enterApp`) ведёт на
  `#scrHome`.
- Книга имён: `om_book_<id>`; история звонков: `om_calls_<id>` (per-user).
- Звонок из книги/истории/хаба не открывает чат: `callContact()` запоминает экран
  и возвращает на него после завершения (`endCall`).
- Доступ к контактам устройства — через стандартный Contact Picker API
  (`navigator.contacts.select`), работает в защищённом контексте (HTTPS) на
  поддерживающих браузерах; есть аккуратный фолбэк.
